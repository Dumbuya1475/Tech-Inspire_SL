How to Set Up GitHub

  1. Create a GitHub account at https://github.com
  2. Click on "Sign Up" and follow the steps.
  3. Verify your email address to activate your account.
  4. Create your first repository to store your code.
  5. Download and install Git on your computer.
  6. Open a terminal and set your username and email: 
    
git config --global user.name "YourName"
git config --global user.email "youremail@example.com"

Basic Command Line Commands

  • cd - Change directory
  • ls - List files and folders
  • mkdir - Create a new directory
  • git clone [URL] - Clone a GitHub repository
  • git add . - Stage all changes
  • git commit -m "Your message" - Commit changes with a message
  • git push - Push your changes to GitHub
    • git pull - Fetch and merge changes from the remote repository
