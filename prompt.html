What is the Command Line?

The command line is a text-based interface that allows you to interact with your computer's files and programs. It is powerful and often used by developers to perform various tasks quickly. One popular command line tool for working with Git is Git Bash.

Basic Command Line Commands

  • cd - Change directory. Example: cd Documents

    • $ cd Documents

  • ls - List files and folders in the current directory. Example: ls

    • $ ls

  • mkdir - Create a new directory. Example: mkdir my-folder

    • $ mkdir my-folder

  • touch - Create a new file. Example: touch file.txt

    • $ touch file.txt

  • rm - Delete a file. Example: rm file.txt

    • $ rm file.txt

  • clear - Clear the terminal screen.

    • $ clear

    Using Git Bash for Git

    Git Bash is a command line tool that lets you use Git commands. Here are some common Git commands to get started:

    $ git init

    $ git clone https://github.com/your-username/your-repository.git

    $ cd your-repository

    $ git add .

    $ git commit -m "Added new feature"

    $ git push

    $ git status
